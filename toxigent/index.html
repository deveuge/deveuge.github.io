Aplicacin web creada con React para mostrar la informacin sobre plantas venenosas para mascotas recopilada por mi web scraper, Poisonous Plants Scraper.

Toxigent

Toxigent es una aplicacin web basada en React y creada con Create React App que permite visualizar informacin sobre plantas que resultan txicas para las mascotas. La informacin es recopilada por un web scraper de desarrollo propio Poisonous Plants Scraper, construido con Puppeteer, que rene informacin sobre plantas venenosas para mascotas y guarda los datos en formato JSON.

Caractersticas

  • Pgina de inicio que enumera todas las plantas venenosas para perros y gatos y le permite filtrar y/o ordenarlas por:

    • Nombre comn o cientfico.
    • Nivel de toxicidad: todos, moderado o grave.
    • Tipo de planta: todas, planta de interior, planta de jardn o planta silvestre.
    • Toxicidad: todos, txico para perros, no txico para perros, txico para gatos, no txico para gatos.
    • Mostrar solo las plantas que contienen informacin detallada: si esta opcin est marcada, solo se mostrarn las plantas que contengan datos en Informacin detallada.

  • Uso de material-table para mostrar todas las plantas en una tabla que ofrece la siguiente funcionalidad:

    • Ordenar por encabezados de tabla.
    • Accin personalizada que permite al usuario navegar al detalle de cada planta.
    • Opcin de exportacin para descargar los datos en formato CSV o PDF.
    • Posibilidad de cambiar al modo nocturno, que cambia la paleta de colores a una ms oscura.
    • Posibilidad de navegar hasta el detalle de una planta aleatoria. Esta opcin se encuentra en la parte inferior de la pgina web.

  • Pgina de detalles de cada planta que muestra la siguiente informacin:

    • Imagen de la planta, obtenida de la informacin detallada. Si no hay una especificado, se mostrar una imagen predeterminada.
    • Icono de la planta.
    • Lista de sntomas comunes.
    • Tipo y toxicidad de la planta.
    • Tabla con informacin detallada de la planta. Si la planta no tiene informacin detallada, se mostrar el mensaje No hay detalles disponibles.
    • Lista de resultados de Wikipedia relacionados con el nombre cientfico de la planta, limitando los resultados a un mximo de 3. Si no se encontraron resultados, se mostrar el mensaje No hay resultados de Wikipedia disponibles.

Aspectos tcnicos

  • Los datos de las plantas se guardan como un archivo JSON en la propia aplicacin para evitar construir un backend que los sirva. La aplicacin carga este fichero al inicio una nica vez y, posteriormente, lo usa para mostrar la informacin solicitada al usuario.
  • No se contemplan libreras de frontend adicionales para construir la interfaz web al considerarlas innecesarias tratndose de una Single Page Application. Es por ello que nicamente se emplea CSS grid y flexbox para hacer un diseo responsivo que se visualice correctamente en todas las pantallas.
